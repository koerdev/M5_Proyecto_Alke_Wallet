# M5 - Proyecto Alke Wallet DB

Repositorio desarrollado para el **Módulo 5: Fundamentos de Bases de datos relacionales**, donde se implementa una base de datos para el proyecto de Alke Wallet.

---

## Tecnologías utilizadas

- **HTML** (estructura y UI básica)  
- **SQL** (consultas sql)

---

## Estructura del proyecto

El proyecto sigue la siguiente estructura:

- PM5/
    - index.html            # Interfaz principal
    - img/                  # Carpeta de imagenes
    - consultas.sql         # Archivo con consultas sql
    - README.md             # Documentación del proyecto

---

## Uso del proyecto

Para usar el proyecto:

1. Clona el repositorio
2. Abre el archivo index.html en tu navegador
3. Sigue las intrucciones de la página para ejecutar las consultas en tu Base de datos.

---

## Consultas implementadas

El proyecto incluye consultas SQL para:

1. Consulta para obtener el nombre de la moneda elegida por un usuario específico.
2. Consulta para obtener todas las transacciones registradas.
3. Consulta para obtener todas las transacciones realizadas por un usuario específico.
4. Sentencia DML para modificar el campo correo electrónico de un usuario específico.
5. Sentencia para eliminar los datos de una transacción (eliminado de la fila completa).
